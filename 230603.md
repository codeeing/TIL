- **프로그램 기본 5대 구성요소**
    1. 변수(variable)
    2. 상수(constant)
    3. 연산자(operator)
    4. 예약어(reserved word)
    5. 함수(function)
    <br><br>
    - *세미콜론은 문장종결자로써 해당되지 않는다.*
<br><br>
- **출력형식, 출력대상, 형식변환문자**

    ``` c
    printf("num = %d\n", 3);
    ```
    - `"num = %d\n"` => 출력형식
    - `%d` => 형식변환문자
    - `3` => 출력대상
<br><br>
- **C언어의 5대 상수**
    1. 숫자상수<br>
    : 정수형상수(10진수, 8진수, 16진수), 실수형상수(10진수)
    2. 문자상수<br>
    : 단일인용부호로 감싼것.&nbsp;&nbsp;ex)&nbsp;'3'
    3. 문자열상수<br>
    : 이중인용부호로 감싼것. 널문자로 문자열을 종료.&nbsp;&nbsp;ex)&nbsp;"3"
    4. 심볼릭상수<br>
    :  매크로상수, 열거형상수, const변수
    - *<u>const를 사용해서 상수화 할 수 있기때문</u>*
    5. 주소상수<br>
    : 변수의 시작 주소, 배열의 시작주소, 함수 각각의 시작 주소
<br><br>
- **상수의 2분류 (기준 : 차원)**
    - 일반상수 : 숫자상수, 문자상수, 매크로상수 일부(#define PI 3.14)<br>
    => 0차원
    - 주소상수 : 문자열상수, 주소상수, 매크로상수 일부 (#define S "smile")<br>
    => 적어도 1차원 이상
<br><br>
- **변수의 2분류 (기준 : 차원)**
    - 일반변수 : 일반상수를 저장<br>
    => 0차원
    - 포인터변수 : 주소상수를 저장<br>
    => 적어도 1차원 이상
<br><br>
- **C언어의 3대 block**
    1. 함수 block
    2. 제어문 block
    3. sub(no name) block
